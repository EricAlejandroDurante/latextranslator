Finalidad

La finalidad del proyecto es romper una de las grandes barreras de la educación en los sectores científicos y matemáticos, en los cuales se trabaja con muchas formulas escritas en código LaTeX, y para que el estudiante no este pasando a mano o hacer esfuerzo para entender el código, decidimos implementar este traductor que recibe el archivo con el código en LaTeX y lo traduce a un archivo con texto plano, para que así el estudiante pueda leer el documento de una manera más simple y sencilla.
